## 1. ゲームコンセプト

### タイトル
魂番長 ～俺たちの青春伝説～ (TAMASHII BANCHOU)

### ジャンル
ヤンキー青春アクションアドベンチャー

### コンセプト
鉄義市（てつぎし）という架空の工業都市を舞台に、転入生の不良少年・神崎 烈が仲間と共に成長し、この街の「真の番長」になるまでの青春譚。

「強さとは何か」「真の仲間とは何か」を問いながら、バカバカしい笑いと胸に刺さる友情、そして手に汗握る喧嘩を体験できる作品。

## 2. 世界観設定

### 舞台：鉄義市（てつぎし）
- 日本の架空都市。かつては製鉄業で栄えた工業都市
- 不景気で荒廃した下町と、新興の繁華街が混在する
- 不良文化・ヤンキー文化が根強く残る
- 複数の高校・グループが縄張り争い

### エリア構成
1. **烈風高校周辺** - スタートエリア。主人公の縄張り
2. **鉄義商店街** - 地元の商店が立ち並ぶ。情報収集の場
3. **港湾区** - 荒くれ者が集まる埠頭エリア
4. **工業区廃工場** - 元製鉄所跡地。ガチンコ勝負の聖地
5. **山上公園** - 市内を一望できる夜景スポット。青春の場
6. **赤羽学園エリア** - ライバル校。入り込むと敵だらけ
7. **鉄義城跡** - 山上の廃城。最終決戦の場

## 4. ストーリー概要

### プロローグ：「鉄義に吹く烈風」
東京から鉄義市に転入してきた神崎烈。早速、地元の不良グループに絡まれるが、烈風高校のヤンキー・武藤剛と乱闘になる。二人の最初の出会いは最悪だったが、そこから物語が始まる。

### 第1章：「烈風高校の掟」
烈風高校に馴染もうとする烈。しかし校内のヒエラルキーは複雑で、様々な先輩・同学年と喧嘩を繰り返す。藤原蓮（元番長）に認められ、烈風高校の正式な一員となる。

### 第2章：「商店街に根付く誇り」
商店街エリアに進出。田中ユキの父・定食屋のオヤジが、港湾区のチンピラグループに因縁をつけられている。烈が介入し、商店街の人々と絆を深める。

### 第3章：「港湾区の狼」
港湾区の支配者・荒波組と対決。このエリアで黒木鷹と出会い、最初は敵として戦うが、共通の敵に立ち向かうことで仲間に。

### 第4章：「影山の影」
鉄義市全体に影山剛臂の支配が及んでいることが判明。赤羽学園も影山の傀儡だった。烈が赤鬼と対決し、その後、赤鬼の真実が明かされ、二人は互いを認め合う。

### 第5章：「決戦！鉄義城跡」
影山剛臂との最終決戦。かつてこの街で誰よりも強かった男との対決。仲間全員の絆を力に、烈が鉄義市の新たな伝説を刻む。

### エピローグ（マルチエンディング）
- **「真番長ルート」**: 全員の絆MAXで真エンディング。鉄義市の学校が一つにまとまる
- **「孤高の拳ルート」**: 黒木鷹との絆がMAXの場合。二人だけで影山を倒す結末
- **「仁義ルート」**: 赤鬼と協力して影山を倒す結末。赤鬼とバディエンド

## 5. システム設計

### 魂ゲージ（Tamashii Gauge）
- 0〜100のゲージ
- 上がる条件：勝利、義理人情行動、弱者保護、啖呵
- 下がる条件：敗北、不義理、逃走
- 高いほどNPCの反応変化、特殊技使用可能

### メンチシステム（Menchi System）
- 喧嘩前の心理戦ミニゲーム
- 相手の目を見つめながらボタンタイミング
- 3ラウンド制。2勝で勝利
- 勝利：先制攻撃+バフ / 敗北：スタン状態でスタート

### 番長ランク（Banchou Rank）
1. チンピラ（Chinpira）
2. 不良（Fûryô）
3. 番長候補（Banchou Kôho）
4. 副番長（Fuku-Banchou）
5. 番長（Banchou）
6. 伝説の番長（Densetsu no Banchou）

### 絆システム（Kizuna System）
- 各仲間と0〜5のランク
- Rank 1: 会話可能
- Rank 2: 同行可能（エリア探索時）
- Rank 3: 合体技解放
- Rank 4: 特別イベント解放
- Rank 5: エンディング分岐条件

### 技システム（Waza System）
**基本技:**
- ジャブ → ストレート → フック（3コンボ）
- アッパー（下段からの打ち上げ）
- ボディブロー

**習得技（烈固有）:**
- 神崎流一本拳（いっぽんけん）: ガード破壊
- 烈風旋回蹴り（れっぷうせんかいげり）: 範囲攻撃
- 魂砕き（たましいくだき）: グラップル技
- 疾風乱打（しっぷうらんだ）: 高速コンボ
- 魂解放（たましいかいほう）: 究極技（魂ゲージ50以上で使用可）

### ファッションシステム
- 学ラン（色・丈変更可）
- 上着（ブレザー/スカジャン/ジャージ）
- 髪型（短髪/リーゼント/モヒカン/ツーブロ）
- アクセサリー（チェーン/バンダナ/指輪）
- ステータスへの影響あり

### ミニゲーム
1. **喧嘩サイコロ（Kenkasaikoro）**: 賭け喧嘩。丁半ゲーム
2. **リズム啖呵（Rhythm Tanka）**: 啖呵を切るリズムゲーム
3. **腕相撲（Arm Wrestling）**: 格闘家との腕相撲
4. **的撃ち（Mato-uchi）**: 缶を石で的撃ち
5. **担いポン（Katsugi Pon）**: おでん屋の屋台担ぎ競争
