# 魂番長 ～俺たちの青春伝説～
## TAMASHII BANCHOU: Legend of Our Youth

> 「男に二言はねえ！」

ヤンキー青春アクションアドベンチャーゲーム。架空の工業都市・鉄義市を舞台に、転入生の神崎烈が真の番長を目指す熱血青春物語。

---

## ゲーム概要

- **ジャンル**: ヤンキー青春アクションアドベンチャー
- **プラットフォーム**: PCブラウザ
- **技術**: Next.js / Three.js / React Three Fiber / Supabase
- **特徴**: メンチシステム / 魂ゲージ / 絆システム / 5章構成ストーリー

## 主な機能

| 機能 | 説明 |
|------|------|
| 🔥 戦闘システム | リアルタイム3D喧嘩。コンボ、ガード、回避、必殺技 |
| 👁️ メンチシステム | 戦闘前の睨み合いミニゲーム。勝利で先制有利 |
| 💪 魂ゲージ | 義理人情の行動でゲージUP。強い技が使えるようになる |
| 🤝 絆システム | 仲間との絆レベル。Rank5で特別展開あり |
| 👘 ファッション | 学ラン・ヘアスタイル・アクセサリーのカスタマイズ |
| ⚔️ 技習得 | レベルアップで新技解放。最大10種類の技 |
| 🎲 ミニゲーム | 喧嘩サイコロ（丁半）・リズム啖呵など |
| 📖 5章ストーリー | マルチエンディング対応の完全オリジナルシナリオ |

## キャラクター

- **神崎 烈** - 主人公。東京からの転入生。熱血漢
- **武藤 剛** - 烈風高校の暴れん坊。最初は敵、後に親友
- **黒木 鷹** - 謎の孤高の格闘家。クールだが熱い
- **藤原 蓮** - 元番長。頭脳派アドバイザー
- **田中 ユキ** - 商店街の定食屋の娘。情報収集担当
- **赤鬼（赤木 鬼太）** - 赤羽学園番長。ラスボス候補
- **影山 剛臂** - 真のラスボス。鉄義市の伝説

## 操作方法

| キー | アクション |
|------|-----------|
| WASD / 矢印 | 移動 |
| Z / J | 軽攻撃（ジャブ等） |
| X / K | 重攻撃（アッパー等） |
| C / L | 必殺技（SP消費） |
| Space | 回避 |
| Shift | ガード |
| R | 挑発（SP回復） |
| Q | 逃げる |
| E / F | NPCに話しかける / 喧嘩を売る |
| M / Esc | メニュー |

## ローカル起動

```bash
npm install
cp .env.example .env.local
# .env.local にSupabaseの設定を記載
npm run dev
```

## テスト

```bash
npm test
```

## ビルド

```bash
npm run build
```
